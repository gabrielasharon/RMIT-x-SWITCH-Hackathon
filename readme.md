Student Experience Web Application
Description
This web application is designed to enhance the experience of students at RMIT University by providing a comprehensive platform for accessing university services, including jobs, events, amenities, facilities, and a leaderboard system to track student activities.
Features
Networking & Events: Allows students to explore and participate in upcoming events.
Leaderboard: Tracks and displays student scores based on their participation and networking.
Job Portal: Lists available part-time and casual job opportunities.
Facilities & Discounts: Displays university facilities and discounts for students.
Chatbot: Offers instant responses to student queries.
Feedback: Students can provide feedback to improve the platform.
Technologies Used
Frontend: Figma (for UI design)
Backend: Django (initially intended but incomplete)
Database: MySQL
Languages: Python, HTML, CSS
Usage
Navigate to the login page to authenticate.
After logging in, explore the dashboard for features like Jobs, Networking, and Leaderboard.
Access amenities, groups, and free tools directly from the dashboard.
